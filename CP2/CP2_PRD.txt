1. Project Overview
Project: CP-2 — Single-Cell Transcriptomic Re-analysis of Myopic Retina and Sclera
Goal: Publicly available scRNA-seq datasets를 통합 재분석하여, OPN5-Dopamine-Hippo-YAP-Sclera cascade를 cell-type resolution으로 검증
Duration: 2–3개월 (8–12주)
Environment: Local Python (로컬 머신/워크스테이션 GPU 권장)
Impact: 중앙 가설의 세포 수준 근거 — IOVS 또는 MedComm 타겟
Cost: $0 (all data and tools are free/open-source)

3.2 GEO Accession 확인 방법
각 논문의 Data Availability 섹션에 GEO accession number가 명시되어 있습니다. 확인 절차:
PubMed에서 논문 검색: 예) "Yao myopia single-cell MedComm 2023"
논문 내 “Data Availability” 또는 “Data Accession” 섹션에서 GSE_______ 확인
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE_______ 로 접속하여 supplementary files 확인
10x Genomics 형식이면 barcodes.tsv.gz, features.tsv.gz, matrix.mtx.gz 세 파일이 있음

4. Step 2: Quality Control & Preprocessing
4.1 QC 메트릭 계산
import scanpy as sc
import numpy as np
import matplotlib.pyplot as plt

sc.settings.verbosity = 3
sc.settings.set_figure_params(dpi=120, facesize=(5,4))

# 데이터 로드 (accession 확인 후 수정)
adata = sc.read_10x_h5('data/raw/yao2023/filtered_feature_bc_matrix.h5')
adata.var_names_make_unique()

# Mitochondrial gene 표시 (mouse = 'mt-', human = 'MT-')
adata.var['mt'] = adata.var_names.str.startswith('mt-')
# Ribosomal gene
adata.var['ribo'] = adata.var_names.str.startswith(('Rps', 'Rpl'))

sc.pp.calculate_qc_metrics(
    adata, qc_vars=['mt', 'ribo'],
    percent_top=None, log1p=False, inplace=True
)
4.2 QC 기준 설정
아래 그래프를 먼저 그려 데이터 분포를 확인한 후 기준을 조정합니다:
# QC violin plot — 분포 확인
sc.pl.violin(adata, ['n_genes_by_counts', 'total_counts', 'pct_counts_mt'],
             jitter=0.4, multi_panel=True)
plt.savefig('figures/qc_violin.png', dpi=150, bbox_inches='tight')

# Scatter plot — outlier 확인
sc.pl.scatter(adata, x='total_counts', y='pct_counts_mt')
sc.pl.scatter(adata, x='total_counts', y='n_genes_by_counts')
일반적 망막 scRNA-seq QC 기준 (데이터 분포 확인 후 조정):
# 필터링 적용
print(f'Before filtering: {adata.n_obs} cells')

adata = adata[adata.obs.n_genes_by_counts > 200, :]
adata = adata[adata.obs.n_genes_by_counts < 6000, :]
adata = adata[adata.obs.pct_counts_mt < 15, :]
adata = adata[adata.obs.total_counts > 500, :]

print(f'After filtering: {adata.n_obs} cells')
⚠ Doublet detection을 추가로 수행하면 데이터 품질이 향상됩니다: pip install scrublet 후 sc.external.pp.scrublet(adata)
4.3 Normalization & HVG Selection
# Raw counts 보존 (나중에 DE 분석용)
adata.layers['counts'] = adata.X.copy()

# Normalization
sc.pp.normalize_total(adata, target_sum=1e4)
sc.pp.log1p(adata)

# Highly Variable Genes
sc.pp.highly_variable_genes(adata, n_top_genes=3000, flavor='seurat_v3',
                            layer='counts')  # seurat_v3는 raw counts 필요

# HVG plot 확인
sc.pl.highly_variable_genes(adata)
plt.savefig('figures/hvg.png', dpi=150, bbox_inches='tight')
4.4 Dimensionality Reduction & Clustering
# Scale (HVG만)
sc.pp.scale(adata, max_value=10)

# PCA
sc.tl.pca(adata, svd_solver='arpack', n_comps=50)
sc.pl.pca_variance_ratio(adata, n_pcs=50)  # elbow plot → n_pcs 결정

# Neighbors + UMAP
sc.pp.neighbors(adata, n_pcs=30)  # elbow plot 기준으로 조정
sc.tl.umap(adata)

# Leiden clustering
sc.tl.leiden(adata, resolution=0.8)  # 세포 유형 수에 따라 0.5–1.5 조정

# UMAP 시각화
sc.pl.umap(adata, color=['leiden'], legend_loc='on data')
plt.savefig('figures/umap_clusters.png', dpi=150, bbox_inches='tight')

# 조건별 세포 유형별 DE (예: RGC에서 myopic vs normal)
rgc = adata[adata.obs['cell_type'] == 'RGC'].copy()

# Wilcoxon rank-sum test (Scanpy default)

8. Step 6: Pseudotime 분석 (공막 Fibroblast Trajectory)
근시 공막에서 fibroblast가 myofibroblast로 전환되는 과정을 pseudotime으로 추적하고, Hippo-YAP 상태를 오버레이합니다:
# 공막 데이터셋 (Wu et al.)에서 fibroblast subset
sclera = adata_sclera[adata_sclera.obs['cell_type'].isin(
    ['Fibroblast', 'Myofibroblast'])].copy()

# Diffusion map
sc.pp.neighbors(sclera, n_pcs=20)
sc.tl.diffmap(sclera)

# PAGA (partition-based graph abstraction)
sc.tl.paga(sclera, groups='cell_type')
sc.pl.paga(sclera)

# DPT (diffusion pseudotime)
sclera.uns['iroot'] = np.flatnonzero(
    sclera.obs['cell_type'] == 'Fibroblast')[0]
sc.tl.dpt(sclera)

# Pseudotime 위에 Hippo-YAP 유전자 overlay
sc.pl.scatter(sclera, basis='diffmap', color=['dpt_pseudotime',
    'Yap1', 'Lats1', 'Tgfb1', 'Col1a1', 'Col3a1', 'Lox', 'Mmp2'],
    ncols=4)
plt.savefig('figures/pseudotime_hippo_yap.png', dpi=200, bbox_inches='tight')

# Pseudotime에 따른 유전자 발현 추이 (smoothed)
sc.pl.paga_path(sclera, nodes=['Fibroblast', 'Myofibroblast'],
    keys=['Yap1','Lats1','Tgfb1','Col1a1','Col3a1','Lox','Hif1a'])
⚠ Pseudotime에서 YAP1이 nuclear (활성)상태로 전환되는 시점과 TGFβ1/MMP 상승 시점이 일치하면, TGFβ switching 가설의 강력한 근거입니다.
9. Step 7: Multi-Dataset Integration
다수 데이터셋을 통합할 때 batch effect를 보정합니다:
import harmonypy as hm

# 데이터셋 병합
adata_combined = adata_yao.concatenate(adata_mrca,
    batch_categories=['Yao2023', 'MRCA2024'])

# 전처리
sc.pp.normalize_total(adata_combined, target_sum=1e4)
sc.pp.log1p(adata_combined)
sc.pp.highly_variable_genes(adata_combined, n_top_genes=3000,
                            batch_key='batch')
sc.pp.scale(adata_combined, max_value=10)
sc.tl.pca(adata_combined, n_comps=50)

# Harmony batch correction
ho = hm.run_harmony(adata_combined.obsm['X_pca'],
                    adata_combined.obs, 'batch')
adata_combined.obsm['X_pca_harmony'] = ho.Z_corr.T

# Harmony PC로 UMAP
sc.pp.neighbors(adata_combined, use_rep='X_pca_harmony')
sc.tl.umap(adata_combined)
sc.pl.umap(adata_combined, color=['batch', 'cell_type'])
